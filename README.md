## Repository Structure

### üìÅ `Data/`
This folder contains data used in the simulation studies and demonstrations reported in the manuscript.

- **Raw simulation data**: Replication-level data generated during simulations.
- **Synthetic demonstration data**:
  - `PIRLS_synthetic_data`  
    A fully synthetic dataset created solely for demonstration purposes.  

---

### üìÑ Appendix Files
All supplemental appendices are named with the prefix **`Appendix_`** and correspond directly to appendices referenced in the manuscript.

- `Appendix_1.pdf`  
  Additional results for Unconstrained Product Indicator (UPI) using all pairs of product indicators.

- `Appendix_2.pdf`  
  Supplementary plots for simulation results.

- `Appendix_3_sim_script.R`  
  R script used to generate simulation data and conduct analyses.

- `Appendix_4_2S-PA-Int_Demonstration_*.pdf`  
  Step-by-step demonstration of the proposed Two-Stage Path Analysis with Interaction (2S-PA-Int) method using synthetic data.

## Software and Reproducibility

Analyses were conducted primarily in **R**.  
Key packages and versions are documented within the scripts themselves.

All data required to reproduce the results reported in the manuscript are included in this repository.
